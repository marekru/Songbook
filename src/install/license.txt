READ THIS FIRST:
Before running this installer, you must already have the following software installed. (If you don't, exit this installer now by clicking the Cancel button, download and install the necessary software, then rerun this installer.)

(1) The MiKTeX typesetting system (a version of LaTeX). The latest version can be downloaded for free from http://www.miktex.org.
(2) The Vim file editor program. The latest version can be download for free from http://www.vim.org.
(3) Any pdf viewer. Adobe Acrobat Reader can be downloaded for free from http://www.adobe.com.

If you choose to continue this installation without installing the above, many important features will be omitted from the installation.
